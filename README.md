# NBTExplorer 3

NBTExplorer 3 是一款基于 NBTExplorer 2.0 的 [Minecraft](https://www.minecraft.net) NBT编辑器，现在采用 WPF 技术重新开发。

## 主要特点
* 全新的 WPF 界面，提供更现代、更流畅的用户体验
* 完整的 NBT 编辑功能，支持所有标准 NBT 操作
* 多文件支持，可以同时打开多个 NBT 文件和文件夹
* 强大的搜索功能，支持按名称和值搜索
* 支持拖放操作，方便文件和节点管理

## 支持的格式
* Standard NBT files (e.g. level.dat)
* Schematic files
* Uncompressed NBT files (e.g. idcounts.dat)
* Minecraft region files (*.mcr)
* Minecraft anvil files (*.mca)
* Cubic Chunks region files (r2*.mcr, r2*.mca)

## 系统要求
Windows 7 或更高版本，.NET 9.0 或更高版本。

## 开发说明
* WPF 版本是当前主要开发版本
* 原始 Windows Forms 版本已不再积极开发
* 所有新功能和 bug 修复将优先在 WPF 版本中实现
